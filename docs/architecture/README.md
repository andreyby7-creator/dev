Детальные архитектурные отчеты по всем компонентам системы SaleSpot BY.

- **disaster-recovery-architecture-report.md** - Архитектура аварийного восстановления

- **Сетевая инфраструктура** - Zero Trust, многоуровневая защита
- **DevOps** - Blue-Green деплойменты, автоматизация
